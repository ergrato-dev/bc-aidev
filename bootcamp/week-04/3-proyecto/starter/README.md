# CLI Utils Package - Starter

Paquete de utilidades de línea de comandos.

## Instalación

```bash
# Crear entorno virtual
python -m venv .venv
source .venv/bin/activate  # Linux/Mac

# Instalar en modo desarrollo
pip install -e .
```

## Uso

```bash
# Ver ayuda
cli-utils --help

# Contar líneas
cli-utils count-lines archivo.txt

# Info del sistema
cli-utils system-info
```
